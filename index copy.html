Courses

This year Belmont School of Programming is offering two courses: Python 101 & Python 102
Python 101 is the course covering all programming fundamentals required to start programming
Python 102 is the subsequent course, that will focus on algorithms and advanced datastructures built from scratch.
Both courses will consist of 10 classes that will both be held over a google meet.

Python 101

Topics our course will cover in 10 classes:
Python 101 will be the introductory class that takes 10 days to complete and will all fundamentals required to begin programming in python. Python 101 is perfect for
Middle - Highschool, but all ages are welcome.
  • Variables
  • Operators
  • Conditions
  • Loops
  • String Operations
  • Loop Operations
  • Data-Types and Structures
  • OOP
59.99$ for 10 hours

Python 102

Topics our course will cover in 10 classes
Python 102 is the subsequent class to Python 101. Python 102 will focus on more conceptual programming on topics such as algorithms and datastructures that will be built from scratch.
  • Lists and Arrays
  • Hash Table
  • Trees
  • Queue
  • Stacks
  • Heaps
  • Graphs
  • Sorting Algorithms
  • Traversals
  • Hashing
  • Dijkstra's Algorithm
  • Big-O runtime
59.99$ for 10 hours
